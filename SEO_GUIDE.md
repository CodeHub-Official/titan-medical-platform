# دليل تحسين محركات البحث (SEO)

## مقدمة

هذا الدليل يشرح كيفية تحسين موقع دليل العيادات الطبي لمحركات البحث (SEO) وزيادة عدد الزوار من Google والمحركات الأخرى.

## 1. البنية الأساسية للموقع

### 1.1 الهيكل الهرمي
```
الرئيسية (/)
├── العيادات (/clinics)
│   ├── القاهرة (/clinics/cairo)
│   └── الجيزة (/clinics/giza)
├── التخصصات (/specialties)
│   ├── تجميل (/specialty/cosmetics)
│   ├── جلدية (/specialty/dermatology)
│   └── ليزر (/specialty/laser)
└── العيادة الفردية (/clinic/[id])
```

### 1.2 روابط داخلية قوية
- كل صفحة عيادة ترتبط بـ:
  - صفحة المدينة
  - صفحة التخصص
  - عيادات مشابهة
  - صفحات ذات صلة

## 2. Meta Tags والبيانات الوصفية

### 2.1 Title Tags
**الطول المثالي**: 50-60 حرف

```
❌ خطأ: عيادة نور
✅ صحيح: عيادة نور - تجميل في التجمع الخامس | دليل العيادات
```

### 2.2 Meta Descriptions
**الطول المثالي**: 150-160 حرف

```
❌ خطأ: عيادة متخصصة
✅ صحيح: عيادة نور متخصصة في جراحات التجميل والليزر بالتجمع الخامس. تقييم 4.8/5 من 150 عميل. اتصل الآن.
```

### 2.3 Keywords
اختر 3-5 كلمات مفتاحية ذات صلة:
- الكلمة الأساسية: "عيادة تجميل في القاهرة"
- كلمات ثانوية: "جراحات تجميل"، "ليزر"، "حقن الفيلر"

## 3. Schema.org Markup

### 3.1 MedicalClinic Schema
```json
{
  "@context": "https://schema.org",
  "@type": "MedicalClinic",
  "name": "عيادة نور",
  "description": "عيادة متخصصة في التجميل",
  "url": "https://medical-directory.com/clinic/clinic-001",
  "telephone": "16226",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "التجمع الخامس",
    "addressLocality": "القاهرة",
    "addressCountry": "EG"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": 4.8,
    "reviewCount": 150
  }
}
```

### 3.2 LocalBusiness Schema
يساعد Google على فهم معلومات العيادة المحلية.

### 3.3 BreadcrumbList Schema
يحسن التنقل والفهم الهرمي للموقع.

## 4. محتوى محسّن للـ SEO

### 4.1 كتابة المحتوى
- استخدم الكلمات المفتاحية بشكل طبيعي
- اكتب محتوى فريد وقيّم
- استخدم العناوين بشكل صحيح (H1, H2, H3)
- اكتب فقرات قصيرة وسهلة الفهم

### 4.2 الكلمات المفتاحية
```
الرئيسية: "عيادات تجميل في القاهرة"
ثانوية:
- عيادات تجميل في التجمع الخامس
- أفضل عيادات تجميل
- جراحات تجميل بالقاهرة
- عيادة تجميل متخصصة
```

### 4.3 محتوى SEO الديناميكي
كل صفحة تحتوي على:
- وصف فريد للعيادة
- معلومات عن التخصص
- معلومات عن المدينة
- أسئلة شائعة (FAQ)

## 5. الصور والوسائط

### 5.1 تحسين الصور
```html
<!-- ❌ خطأ -->
<img src="image.jpg" alt="صورة">

<!-- ✅ صحيح -->
<img 
  src="clinic-nour-cairo.jpg" 
  alt="عيادة نور - تجميل في التجمع الخامس"
  title="عيادة نور"
>
```

### 5.2 حجم الصور
- استخدم صور بحجم 1200x630 للـ Open Graph
- استخدم صور بحجم 600x400 للقوائم
- استخدم WebP للأداء الأفضل

### 5.3 Lazy Loading
```html
<img src="..." loading="lazy" />
```

## 6. الأداء والسرعة

### 6.1 Core Web Vitals
- **LCP**: < 2.5s (Largest Contentful Paint)
- **FID**: < 100ms (First Input Delay)
- **CLS**: < 0.1 (Cumulative Layout Shift)

### 6.2 تحسين السرعة
- استخدم Static Generation
- فعّل Caching
- استخدم CDN
- ضغط الصور
- Minify CSS و JavaScript

## 7. Mobile Optimization

### 7.1 Responsive Design
- اختبر على أجهزة مختلفة
- استخدم Tailwind CSS للتجاوب
- اختبر سرعة الموبايل

### 7.2 Mobile-First Indexing
Google يفهرس الموقع من نسخة الموبايل أولاً.

## 8. Sitemap و Robots.txt

### 8.1 Sitemap.xml
- يتم توليده تلقائياً
- يحتوي على جميع الصفحات
- يتم تحديثه بانتظام

### 8.2 Robots.txt
```
User-agent: *
Allow: /
Disallow: /admin

Sitemap: https://medical-directory.com/sitemap.xml
```

## 9. Google Search Console

### 9.1 التحقق من الملكية
1. اذهب إلى https://search.google.com/search-console
2. أضف الموقع
3. تحقق من الملكية (HTML tag أو DNS)

### 9.2 إرسال Sitemap
1. اذهب إلى "Sitemaps"
2. أرسل `sitemap.xml`

### 9.3 مراقبة الأداء
- راقب الكلمات المفتاحية
- راقب نسبة النقر
- راقب الأخطاء

## 10. Google My Business

### 10.1 إنشاء قائمة
1. اذهب إلى https://business.google.com
2. أنشئ قائمة لكل عيادة
3. أضف معلومات كاملة

### 10.2 المعلومات الضرورية
- الاسم
- العنوان
- رقم الهاتف
- ساعات العمل
- الفئة (Medical Clinic)
- الصور
- الموقع

## 11. الروابط الخارجية (Backlinks)

### 11.1 بناء الروابط
- اطلب من العيادات الربط إلى صفحاتهم
- اكتب مقالات ضيف
- استخدم الدلائل المحلية

### 11.2 جودة الروابط
- روابط من مواقع موثوقة أفضل من الكثير من الروابط الضعيفة
- تجنب الروابط المشبوهة

## 12. المحتوى المحلي

### 12.1 تحسين المحتوى المحلي
- استخدم أسماء المدن والأحياء
- اذكر المعالم المحلية
- استخدم الأرقام المحلية

### 12.2 الكلمات المفتاحية المحلية
```
- عيادات في القاهرة
- أفضل تجميل بالتجمع الخامس
- جلدية في مصر الجديدة
```

## 13. الكلمات المفتاحية طويلة الذيل

### 13.1 أمثلة
```
- "عيادة تجميل في التجمع الخامس بالقاهرة"
- "أفضل عيادة جلدية في مصر الجديدة"
- "ليزر إزالة الشعر في الشيخ زايد"
```

## 14. الأسئلة الشائعة (FAQ)

### 14.1 FAQ Schema
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "ما هي خدمات العيادة؟",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "توفر العيادة خدمات..."
      }
    }
  ]
}
```

### 15.2 المقاييس الرئيسية

## 16. التحديثات والصيانة

### 16.1 تحديث المحتوى
- أضف عيادات جديدة بانتظام
- حدّث معلومات العيادات
- أضف مقالات جديدة

### 16.2 مراقبة الأخطاء
- تحقق من الروابط المكسورة
- تحقق من الأخطاء 404
- تحقق من الأخطاء 500

## 17. أفضل الممارسات

### 17.1 ✅ افعل
- استخدم HTTPS
- اكتب محتوى فريد
- استخدم كلمات مفتاحية طبيعية
- اختبر على الموبايل
- استخدم Schema.org
- بناء روابط داخلية قوية

### 17.2 ❌ لا تفعل
- لا تستخدم محتوى مكرر
- لا تحشو الكلمات المفتاحية
- لا تستخدم روابط مشبوهة
- لا تخفي محتوى من المستخدمين
- لا تستخدم Cloaking
- لا تشتري روابط
