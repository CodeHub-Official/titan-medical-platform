### 2. إدارة المقالات

#### إنشاء مقالات جديدة

```typescript
interface Article {
  id: string;                    // معرّف فريد
  title: string;                 // عنوان المقال
  content: string;               // محتوى المقال
  author: string;                // اسم الكاتب
  createdAt: string;             // تاريخ الإنشاء
  images: string[];              // الصور المرفقة
  pageType: 'clinic' | 'specialty' | 'city' | 'general';
  pageId?: string;               // معرّف الصفحة
  status: 'draft' | 'published'; // حالة المقال
  seoTitle: string;              // عنوان SEO
  seoDescription: string;        // وصف SEO
}
```

#### العمليات المدعومة

- ✅ إنشاء مقال جديد
- ✅ تعديل مقال موجود
- ✅ حذف مقال
- ✅ نشر/إخفاء مقال
- ✅ إضافة صور متعددة
- ✅ تحسين SEO
