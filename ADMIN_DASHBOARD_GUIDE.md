## 🎯 نظرة عامة

**لوحة التحكم الإدارية** هي أداة متقدمة لإدارة ونشر وتصدير المقالات من موقع دليل العيادات الطبي. تتيح للمسؤولين إنشاء محتوى غني بالصور والوسائط وتصديره بصيغ متعددة.

### الهدف الرئيسي

توفير واجهة سهلة وقوية لإدارة المحتوى والمقالات مع إمكانية التصدير بصيغ مختلفة (HTML, PDF, Markdown, JSON).

---

## ✨ الميزات الرئيسية

### 1. إدارة المقالات

#### إنشاء مقالات جديدة
- عنوان المقال
- محتوى نصي غني
- اسم الكاتب
- نوع الصفحة (عام، عيادة، تخصص، مدينة)
- حالة المقال (مسودة، منشور)

#### تعديل المقالات
- تحديث جميع معلومات المقال
- إضافة أو حذف الصور
- تحديث محتوى SEO

#### حذف المقالات
- حذف آمن مع تأكيد
- حذف فوري من قاعدة البيانات

### 2. إدارة الصور والوسائط

#### إضافة الصور
- دعم صور متعددة
- تحميل مباشر من الجهاز
- عرض معاينة للصور
- حذف الصور غير المطلوبة

#### معرض الصور
- عرض جميع الصور المرفقة
- معاينة سريعة
- حذف سهل

### 3. تحسين محركات البحث (SEO)

#### Meta Tags
- عنوان SEO (حتى 60 حرف)
- وصف SEO (حتى 160 حرف)
- عداد الأحرف التلقائي

#### معلومات إضافية
- نوع الصفحة
- معرّف المقال
- تاريخ الإنشاء

### 4. تصدير المقالات

#### صيغ التصدير المدعومة

**HTML**
- تصميم احترافي
- دعم الصور
- معلومات SEO
- جاهز للطباعة
- جاهز لتحويل PDF

**PDF**
- يتم من خلال طباعة HTML إلى PDF
- تصميم متجاوب
- دعم الصور الكاملة

**Markdown**
- صيغة نصية بسيطة
- جداول معلومات
- دعم الصور
- سهل التعديل

**JSON**
- صيغة منظمة
- جميع البيانات الكاملة
- سهل الاستيراد
- جاهز للـ API

## 🔗 كيفية الوصول

### من المتصفح

```
http://localhost:3000/admin
```

1. اذهب إلى الصفحة الرئيسية
2. انقر على زر "🎛️ لوحة التحكم" في رأس الصفحة

1. افتح الموقع على هاتفك
2. انقر على قائمة الهاتف (☰)
3. اختر "🎛️ لوحة التحكم"

## 📝 إدارة المقالات

### إنشاء مقال جديد

#### الخطوة 1: فتح نموذج الإضافة
```
انقر على زر "➕ إضافة مقال جديد"
```

#### الخطوة 2: ملء البيانات الأساسية

| الحقل | الوصف | مثال |
|------|-------|------|
| **عنوان المقال** | العنوان الرئيسي | "فوائد عيادة الأسنان الحديثة" |
| **اسم الكاتب** | من كتب المقال | "د. أحمد محمد" |
| **نوع الصفحة** | نوع المحتوى | عيادة / تخصص / مدينة |
| **حالة المقال** | منشور أم مسودة | مسودة / منشور |

#### الخطوة 3: كتابة المحتوى

```
اكتب محتوى المقال في حقل "محتوى المقال"
يمكنك استخدام:
- فقرات منفصلة
- نقاط رئيسية
- معلومات تفصيلية
```

#### الخطوة 4: إضافة معلومات SEO

```
عنوان SEO: "أفضل عيادات الأسنان في القاهرة"
وصف SEO: "اكتشف أفضل عيادات الأسنان المتخصصة..."
```

#### الخطوة 5: إضافة الصور

```
1. انقر على "📸 إضافة صور"
2. اختر صور من جهازك
3. سيتم عرض معاينة الصور
4. يمكنك حذف أي صورة غير مطلوبة
```

#### الخطوة 6: حفظ المقال

```
انقر على "✅ إضافة المقال"
سيتم حفظ المقال فوراً
```

### تعديل مقال موجود

#### الخطوة 1: اختيار المقال
```
انقر على بطاقة المقال المراد تعديله
أو انقر على زر "✏️ تعديل"
```

#### الخطوة 2: تحديث المعلومات
```
عدّل أي معلومة تريدها
أضف أو احذف الصور
حدّث محتوى SEO
```

#### الخطوة 3: حفظ التغييرات
```
انقر على "💾 تحديث المقال"
سيتم حفظ التغييرات فوراً
```

### حذف مقال

#### الخطوة 1: اختيار المقال
```
انقر على زر "🗑️ حذف" في بطاقة المقال
```

#### الخطوة 2: تأكيد الحذف
```
سيظهر تأكيد: "هل أنت متأكد من حذف هذا المقال؟"
انقر على "نعم" للتأكيد
```

#### الخطوة 3: تم الحذف
```
سيتم حذف المقال فوراً من قائمة المقالات
```

## 📥 تصدير المقالات

### تصدير إلى HTML

#### الخطوات

1. اذهب إلى تبويب "📥 التصدير"
2. اختر المقال المراد تصديره
3. انقر على زر "📄 HTML"
4. سيتم تحميل ملف HTML

#### استخدام الملف

```html
<!-- فتح الملف في المتصفح -->
<!-- أو طباعته إلى PDF -->
<!-- أو نشره على الويب -->
```

#### المميزات

- ✅ تصميم احترافي
- ✅ دعم الصور الكاملة
- ✅ معلومات SEO مضمنة
- ✅ جاهز للطباعة
- ✅ جاهز لتحويل PDF

### تصدير إلى PDF

#### الخطوات

1. صدّر المقال إلى HTML أولاً
2. افتح الملف في المتصفح
3. اضغط Ctrl+P (أو Cmd+P على Mac)
4. اختر "حفظ كـ PDF"
5. اختر المجلد والاسم
6. انقر "حفظ"

#### الإعدادات الموصى بها

```
الهوامش: 2 سم
الحجم: A4
الاتجاه: عمودي
الخلفيات: مفعلة
```

### تصدير إلى Markdown

#### الخطوات

1. اذهب إلى تبويب "📥 التصدير"
2. اختر المقال
3. انقر على زر "📝 Markdown"
4. سيتم تحميل ملف .md

#### استخدام الملف

```markdown
# استخدم الملف في
- GitHub
- GitLab
- Notion
- أي محرر Markdown
```

#### المميزات

- ✅ صيغة نصية بسيطة
- ✅ سهل التعديل
- ✅ دعم الجداول
- ✅ دعم الصور
- ✅ قابل للمشاركة

### تصدير إلى JSON

#### الخطوات

1. اذهب إلى تبويب "📥 التصدير"
2. اختر المقال
3. انقر على زر "⚙️ JSON"
4. سيتم تحميل ملف .json

#### استخدام الملف

```json
{
  "id": "article-1",
  "title": "عنوان المقال",
  "content": "محتوى المقال",
  "images": ["url1", "url2"],
  ...
}
```

#### المميزات

- ✅ جميع البيانات الكاملة
- ✅ سهل الاستيراد
- ✅ جاهز للـ API
- ✅ منظم وواضح

## 💡 الأمثلة العملية

### مثال 1: إنشاء مقال عن عيادة

```
1. اضغط "➕ إضافة مقال جديد"

2. ملء البيانات:
   - العنوان: "عيادة الأسنان المتقدمة - الجيزة"
   - الكاتب: "د. محمد علي"
   - النوع: "عيادة"
   - الحالة: "منشور"

3. المحتوى:
   "تتخصص عيادتنا في...
   نقدم خدمات...
   الفريق الطبي..."

4. SEO:
   - العنوان: "أفضل عيادة أسنان في الجيزة"
   - الوصف: "عيادة متخصصة في طب الأسنان..."

5. الصور:
   - أضف 3-5 صور للعيادة

6. احفظ المقال
```

### مثال 2: تصدير مقال إلى PDF

```
1. اذهب إلى تبويب "📥 التصدير"

2. اختر المقال

3. انقر "📄 HTML"

4. افتح الملف في المتصفح

5. اضغط Ctrl+P

6. اختر "حفظ كـ PDF"

7. اختر المجلد والاسم

8. انقر "حفظ"
```

### مثال 3: تحديث مقال موجود

```
1. اضغط على بطاقة المقال

2. انقر "✏️ تعديل"

3. حدّث العنوان أو المحتوى

4. أضف صور جديدة إذا لزم

5. حدّث معلومات SEO

6. انقر "💾 تحديث المقال"
```

## 🆘 استكشاف الأخطاء

### المشكلة: الصور لا تظهر

```
الحل:
1. تأكد من أن الملف صورة فعلاً
2. تحقق من حجم الملف
3. جرب صورة أخرى
4. أعد تحميل الصفحة
```

### المشكلة: لا يمكن حفظ المقال

```
الحل:
1. تأكد من ملء جميع الحقول المطلوبة
2. تحقق من الاتصال بالإنترنت
3. أعد تحميل الصفحة
4. جرب متصفح آخر
```

### المشكلة: التصدير لا يعمل

```
الحل:
1. تأكد من أن لديك مقالات
2. تحقق من إعدادات المتصفح
3. أعد تحميل الصفحة
4. جرب متصفح آخر
```
