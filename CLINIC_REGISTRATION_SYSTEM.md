## 🔄 إدارة الاشتراكات

### تتبع الاشتراكات

```sql
-- جدول الاشتراكات
CREATE TABLE subscriptions (
  id UUID PRIMARY KEY,
  clinic_id UUID REFERENCES clinics(id),
  plan_type VARCHAR(50), -- free, basic, premium, annual
  status VARCHAR(50), -- active, expired, cancelled
  start_date TIMESTAMP,
  end_date TIMESTAMP,
  auto_renew BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP
);
```

### تجديد الاشتراكات

```typescript
// فحص الاشتراكات المنتهية يومياً
export async function checkExpiredSubscriptions() {
  const { data: expired } = await supabase
    .from('subscriptions')
    .select('*')
    .lt('end_date', new Date().toISOString())
    .eq('status', 'active')

  // تحديث الحالة
  for (const sub of expired) {
    await supabase
      .from('subscriptions')
      .update({ status: 'expired' })
      .eq('id', sub.id)

    // تحديث حالة العيادة
    await supabase
      .from('clinics')
      .update({ subscription_type: 'free' })
      .eq('id', sub.clinic_id)
  }
}
```

### التجديد التلقائي

```typescript
// تجديد تلقائي قبل انتهاء الاشتراك
export async function autoRenewSubscription(subscriptionId: string) {
  const subscription = await getSubscription(subscriptionId)

  if (subscription.auto_renew) {
    // إنشاء دفعة جديدة
    const payment = await createPayment({
      clinic_id: subscription.clinic_id,
      amount: getPlanPrice(subscription.plan_type),
      subscription_type: subscription.plan_type
    })

    // معالجة الدفع
    await processPayment(payment)
  }
}
```
